# UML图

## 什么是UML？

一种通用可视化建模语言。用来对软件密集型系统进行可视化，详述，构造和文档化

## 分层

- 概念层：一种图形表示方法，让相关人员一眼就看出所要表达的含义
- 规格说明层：一种图形，目的是将该图形转换为程序代码
- 实现层：将规格说明转换为程序代码
